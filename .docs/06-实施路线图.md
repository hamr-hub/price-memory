# 实施路线图（Execution Plan）

## 当前状态（MVP）
- 后端：FastAPI + SQLAlchemy + Playwright，支持 Supabase 回退逻辑
- 前端：React + Refine + AntD，支持商品添加、搜索、任务执行、导出

## 里程碑
- M1：文档与接口统一（已完成）
- M2：调度与重试
  - 新增任务计划字段与执行器（简单轮询/cron）
  - 失败重试与超时控制
- M3：反爬与稳定性
  - UA/时区/网络节流与随机等待
  - 错误分类与告警占位
- M4：容器化与CI
  - 后端与前端镜像
  - 构建与测试流水线
- M5：数据指标与报表
  - 波动统计、区间分布、导出增强

## 风险与应对
- 站点反爬：减少请求频率、随机化、解析兜底
- 云端不可用：本地数据库回退与导出可用
- 浏览器安装失败：Playwright 安装与缓存镜像

## 验收标准
- 接口稳定且与文档一致
- 关键路径：添加商品→执行任务→查看历史→导出可用
- 失败路径有明确错误信息
