# Price Memory 项目概述

## 项目简介

Price Memory（价格记忆）是一个智能价格监控系统，旨在帮助用户追踪商品价格变化，做出更明智的购买决策。系统通过自动化爬虫技术收集商品价格信息，并提供直观的数据分析和可视化界面。

## 核心功能

### 🕷️ 智能爬虫
- 基于 Playwright 的高性能网页爬虫
- 支持动态网页内容抓取
- 灵活的数据提取规则配置
- 反反爬虫机制

### 📊 数据分析
- 价格趋势分析
- 历史价格对比
- 价格波动预测
- 统计图表生成

### 💾 数据存储
- PostgreSQL 数据库存储
- Supabase 云端数据库支持
- 数据备份和恢复机制

### 🖥️ 管理界面
- 基于 React + Refine 的现代化管理后台
- 响应式设计，支持多设备访问
- 实时数据更新
- 用户友好的操作界面

## 技术特色

### 🔧 技术栈现代化
- **前端**: TypeScript + React + Vite + Refine + Ant Design

### 🚀 高性能架构
- 微服务架构设计
- 容器化部署
- 水平扩展支持
- 高可用性保障

### 🛡️ 安全可靠

## 应用场景

### 个人用户
- 监控心仪商品价格变化
- 设置价格提醒
- 历史价格查询
- 最佳购买时机分析

### 企业用户
- 竞品价格监控
- 市场价格分析
- 定价策略制定
- 商业智能决策

### 电商平台
- 价格竞争分析
- 动态定价策略
- 市场趋势预测
- 用户行为分析

1. **灵活性**: 支持抓取任何公开的商品信息维度
2. **准确性**: 高精度的数据提取和处理
3. **实时性**: 实时价格监控和更新
4. **易用性**: 直观的用户界面和操作流程
5. **扩展性**: 模块化设计，易于功能扩展
6. **稳定性**: 完善的错误处理和恢复机制
